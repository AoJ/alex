Example call

This is an example of a dialogue where a user is looking for a cheap chinese restaurant. Later, the user changes its mind and wants an expensive restaurant. At the end of the dialogue, the user asks about the address and the phone number.

Speaker Text
System Thank you for calling the Cambridge Information system. Your call will be recorded for research purposes. You may ask for information about a place to eat, a place to drink, somewhere to stay, or any place you would like to see. How may I help you?
User HI I'M LOOKING FOR A CHEAP CHINESE RESTAURANT
System There is no place to eat serving chinese food in the cheap price range.
User I WANT SOMETHING MODERATE
System Blue Spice is a wonderful restaurant in the moderate price range.
User IS IT A CHINESE RESTAURANT
System No. Blue Spice is a wonderful restaurant and it serves indian food.
User I WANT A CHINESE RESTAURANT
System There is no place to eat serving chinese food in the moderate price range.
User I'D LIKE SOMETHING EXPENSIVE
System The Peking Restaurant, which is in the expensive price range, is a wonderful restaurant. It serves chinese food.
User DO YOU HAVE ANYTHING ELSE
System The Agora serves turkish food and it's a wonderful restaurant in the central part of town near King's College, also I am sorry but there is no information about the price range for it.
User I'M LOOKING FOR A CHINESE RESTAURANT
System Yu Garden serves chinese food. It's in the Fen Ditton part of town and it's a wonderful restaurant near Taxi Gallery, also I am sorry but there is no information about the price range for it.
User OK GIVE ME THE ADDRESS
System Yu Garden is located on A/529 Newmarket Road.
User AND A PHONE NUMBER
System The phone number of Yu Garden is 01223248882.
User THANK YOU GOODBYE
System Thank you for using this system. Could you tell us whether you were satisfied, by saying yes, or no?
User YES
System Thank you, goodbye
